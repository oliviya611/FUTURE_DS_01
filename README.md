# 📊 Marketing Funnel & Conversion Analysis

## 🎯 Objective
Analyze marketing funnel performance to identify conversion rates, drop-offs, and improvement opportunities.

---

## 📁 Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

---

## 🔄 Funnel Stages
Contacts → Leads → Customers

---

## 📈 Key Results
- Low overall conversion rate
- High drop-off at lead-to-customer stage
- Some customer segments perform better than others

---

## 💡 Recommendations
- Improve targeting strategy
- Personalize marketing campaigns
- Strengthen follow-up process
- Optimize campaign performance

---

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
